Field/Component Field Name FIXML name Req'd Comments Depr.
534 NoAffectedOrders @NoAffctdOrds Optional field used to indicate the number of order identifiers for orders affected by the mass action request. Must be followed with OrigClOrdID (41) as the next field.
41 OrigClOrdID @OrigClOrdID Required if NoAffectedOrders > 0 and must be the first repeating field in the group. Indicates the client order id of an order affected by this request. If order(s) were manually delivered (or otherwise not delivered over FIX and not assigned a ClOrdID) this field should contain string "MANUAL".
535 AffectedOrderID @AffctdOrdID Contains the OrderID assigned by the counterparty of an affected order. Not required as part of the repeating group if OrigClOrdID(41) has a value other than "MANUAL".
536 AffectedSecondaryOrderID @AffctdScndOrdID Contains the SecondaryOrderID assigned by the counterparty of an affected order. Not required as part of the repeating group
