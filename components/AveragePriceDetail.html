FIX.5.0SP2 Components

AveragePriceDetail [type 'Component']

The AveragePriceDetail component provides average pricing details in a trade report, including the average pricing model and the start and end times of averaging period.

Field/Component Field Name FIXML name Req'd Comments Depr.
2763 AveragePriceType @Typ
2764 AveragePriceStartTime @StartTm Required if AveragePriceType(2763)=2 (Percent of volume average price) or 0 (Time weighted average price).
2765 AveragePriceEndTime @EndTm Required if AveragePriceType(2763)=2 (Percent of volume average price) or 0 (Time weighted average price).
