FIX.5.0SP2 Components

LegOptionExerciseDateGrp [type 'Component']

The LegOptionExerciseDateGrp is a repeating subcomponent of the LegOptionExerciseDates component used to specify fixed dates for exercise.

Field/Component Field Name FIXML name Req'd Comments Depr.
41512 NoLegOptionExerciseDates
41513 LegOptionExerciseDate @Dt Required if NoLegOptionExerciseDates(41512) > 0.
41514 LegOptionExerciseDateType @Typ When specified it applies not only to the current date but to all subsequent dates in the group until overridden with a new type.
