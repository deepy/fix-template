FIX.5.0SP2 Components

LegPaymentStreamFixingDateGrp [type 'Component']

LegPaymentStreamFixingDateGrp is a subcomponent of the LegPaymentStreamResetDates component used to specify predetermined fixing dates.

Field/Component Field Name FIXML name Req'd Comments Depr.
42459 NoLegPaymentStreamFixingDates
42460 LegPaymentStreamFixingDate @Dt Required if NoLegPaymentStreamFixingDates(42459) > 0.
