FIX.5.0SP2 Components

LegReturnRateValuationDateGrp [type 'Component']

LegReturnRateValuationDateGrp is a repeating subcomponent within the LegReturnRateDateGrp component. It is used to specify the fixed valuation dates for an equity return swap payment stream.

Field/Component Field Name FIXML name Req'd Comments Depr.
42571 NoLegReturnRateValuationDates
42572 LegReturnRateValuationDate @Dt Required if NoLegReturnRateValuationDates(42571) > 0.
42573 LegReturnRateValuationDateType @Typ When specified it applies not only to the current date instance but to all subsequent date instances in the group until overridden when a new type is specified.
