FIX.5.0SP2 Components

MDStatisticReqGrp [type 'Component']

This component block is used within the MarketDataStatisticsRequest(35=DO) message to define a set of parameters describing the desired statistics.

Field/Component Field Name FIXML name Req'd Comments Depr.
2474 NoMDStatistics None
2475 MDStatisticID @StatsID Required if NoMDStatistics(2474) > 0. Unique statistics identifier used as a placeholder for a set of parameters. If an ID is not applicable use "[N/A]".
Component MDStatisticParameters StatsPrm Required if NoMDStatistics(2474) > 0 and MDStatisticID(2475) = "[N/A]".
