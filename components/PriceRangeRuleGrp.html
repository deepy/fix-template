FIX.5.0SP2 Components

PriceRangeRuleGrp [type 'Component']

The PriceRangeRulesGrp component is used to specify the price range rules for a given product group or complex.

Field/Component Field Name FIXML name Req'd Comments Depr.
2550 NoPriceRangeRules None
2551 StartPriceRange @StartPxRng Required if NoPriceRangeRules(2550) > 0.
2552 EndPriceRange @EndPxRng None
2553 PriceRangeValue @PxRngValu Mutually exclusive with PriceRangePercentage(2554).
2554 PriceRangePercentage @PxRngPctage Mutually exclusive with PriceRangeValue(2553).
2556 PriceRangeRuleID @PxRngRuleID Can be used to provide an identifier so that the rule can be reference via the ID elsewhere.
2555 PriceRangeProductComplex @PxRngProdCmplx Can be used to limit price range to specific product suite.
