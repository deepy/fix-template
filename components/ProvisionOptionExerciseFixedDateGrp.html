FIX.5.0SP2 Components

ProvisionOptionExerciseFixedDateGrp [type 'Component']

The ProvisionOptionExerciseFixedDateGrp is a repeating component within the ProvisionOptionExerciseDates component used to report an array of unadjusted or adjusted fixed exercise dates.

Field/Component Field Name FIXML name Req'd Comments Depr.
40142 NoProvisionOptionExerciseFixedDates
40143 ProvisionOptionExerciseFixedDate @Dt Required if NoProvisionOptionExerciseFixedDates (40142) > 0.
40144 ProvisionOptionExerciseFixedDateType @Typ When specified it applies not only to the current date but to all subsequent dates in the group until overridden with a new type.
