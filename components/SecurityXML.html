FIX.5.0SP2 Components

SecurityXML [type 'Component']

The SecurityXML component is used for carrying security description or definition in an XML format. See "Specifying an FpML product specification from within the FIX Instrument Block" for more information on using this component block with FpML as a guideline.

Field/Component Field Name FIXML name Req'd Comments Depr.
1184 SecurityXMLLen Must be set if SecurityXML field is specified and must immediately precede it.
1185 SecurityXML XML payload or content describing the Security information.
1186 SecurityXMLSchema @Schema XML Schema used to validate the XML used to describe the Security.
