SettlParties [type 'Component']

The SettlParties component block is used in a similar manner as Parties Block within the context of settlement instruction messages to distinguish between parties involved in the settlement and parties who are expected to execute the settlement process.

Field/Component Field Name FIXML name Req'd Comments Depr.
781 NoSettlPartyIDs Repeating group below should contain unique combinations of SettlPartyID, SettlPartyIDSource, and SettlPartyRole
782 SettlPartyID @ID Used to identify source of SettlPartyID. Required if SettlPartyIDSource is specified. Required if NoSettlPartyIDs > 0.
783 SettlPartyIDSource @Src Used to identify class source of SettlPartyID value (e.g. BIC). Required if SettlPartyID is specified. Required if NoSettlPartyIDs > 0.
784 SettlPartyRole @R Identifies the type of SettlPartyID (e.g. Executing Broker). Required if NoSettlPartyIDs > 0.
