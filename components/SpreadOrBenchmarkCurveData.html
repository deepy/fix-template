SpreadOrBenchmarkCurveData [type 'Component']

The SpreadOrBenchmarkCurveData component block is primarily used for Fixed Income to convey spread to a benchmark security or curve.

Field/Component Field Name FIXML name Req'd Comments Depr.

663 BenchmarkPriceType @PxTyp Must be present if BenchmarkPrice is used.

761 BenchmarkSecurityIDSource @SecIDSrc Source of BenchmarkSecurityID. If not specified, then ID Source is understood to be the same as that in the Instrument block.
