FIX.5.0SP2 Components

UnderlyingPaymentStreamPaymentDateGrp [type 'Component']

The UnderlyingPaymentStreamPaymentDateGrp is a repeating subcomponent of the UnderlyingPaymentStreamPaymentDates component used to detail fixed dates for swap stream payments.

Field/Component Field Name FIXML name Req'd Comments Depr.
41937 NoUnderlyingPaymentStreamPaymentDates
41938 UnderlyingPaymentStreamPaymentDate @Dt Required if NoUnderlyingPaymentStreamPaymentDates(41937) > 0.
41939 UnderlyingPaymentStreamPaymentDateType @Typ When specified it applies not only to the current date but to all subsequent dates in the group until overridden with a new type.
