OrderResponseLevel [type '2427']

The level of response requested from receiver of mass order messages. A default value should be bilaterally agreed.

Tag Field Name FIXML name Comments Depr.
OrderResponseLevel OrdRspLvl The level of response requested from receiver of mass order messages. A default value should be bilaterally agreed.
0 = NoAck
1 = MinimumAck
2 = AckEach
3 = SummaryAck
