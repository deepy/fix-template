GTBookingInst [type '427']

Code to identify whether to book out executions on a part-filled GT order on the day of execution or to accumulate.

Field/Component Field Name FIXML name Comments Depr.
427 GTBookingInst GTBkngInst Code to identify whether to book out executions on a part-filled GT order on the day of execution or to accumulate.
0 = BookOutAllTradesOnDayOfExecution
1 = AccumulateUntilFilledOrExpired
2 = AccumulateUntilVerballyNotifiedOtherwise
